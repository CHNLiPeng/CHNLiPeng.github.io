title: Hexo配置个人博客

tags: [blog搭建]

Hexo博客搭建教程（Mac用户版）

Hexo的安装

hexo需要依赖两个组件来运行,所以我们要确保自己安装了这两个组件

  1. Git ,我是通过安装Githud desktop版本来安装的。官网下载链接

    在终端里,自己随便创建一个英文名文件夹,然后执行git init方法,看是否执行成功,就能知道自己是否成功安装git

  2. Node.js 官网下载链接

    在终端输入 npm -v就可以查看自己是否成功安装Node.js了如果安装成功，则出现以下情况.

    $ npm -v

    2.11.3

然后，打开终端,在终端里输入以下命令就可以开始安装hexo了。

$ sudo npm install -g hexo-cli

过程中可能会卡在node-gyp rebuild这个阶段，请耐心等待(墙太高,要慢慢爬)

安装成功后,可以通过在终端输入hexo -v查看是否安装成功 

user:~ lipeng$ hexo -v

hexo-cli: 0.1.8

os: Darwin 14.4.0 darwin x64

http_parser: 2.3

node: 0.12.7

v8: 3.28.71.19

uv: 1.6.1

zlib: 1.2.8

modules: 14

openssl: 1.0.1p

这时候hexo的安装就大功告成了。

开始创建博客

  1. 在终端里,进去你电脑里你准备存放你所写文章的目录(最好是英文名字).

    然后在该目录下执行以下命令

    $ hexo init

执行成功后: 

fuzifudeMacBook-Pro:Hexo lipeng$ hexo init

INFO  Copying data to ~/Hexo

INFO  You are almost done! Don't forget to run 'npm install' before you start blogging with Hexo!

这时候你的目录里面会多出很多文件和文件夹。我们可以使用$ ls命令查看到以下情况

$ ls

_config.yml    package.json    scaffolds    source        themes

├── _config.yml

├── package.json

├── scaffolds

├── scripts

├── source

| ├── _drafts

| └── _posts

└── themes

各个目录的功能:

然后继续在本目录下执行以下命令

$ npm install hexo

$ hexo server    

INFO  Hexo is running at http://0.0.0.0:4000/. Press Ctrl+C to     stop.

然后在浏览器输入 http://0.0.0.0:4000/这个网址,就可以看到你自己在本地创建的博客了

将你的博客摆上网络。

使用你的Githud账户建立一个username.github.io的库

例子:我在githud上面的名字叫CHNLiPeng,所以我的应该创建一个名字叫CHNLiPeng.github.io例子连接的仓库。(字母不要写错,最好粘贴我这里的例子过去,然后直接修改成你的githud名)

  • 建立时记得勾选 Initialize this repository with a README这个选项。(其它的不需要修改)

hexo3.0以上的版本需要安装这个文件

$ npm install hexo-deployer-git --save

配置文件里deploy的type改成git(3.0以下的是githud)

如果你安装的时候,卡在了这一步:

> node-gyp rebuild

则你需要npm的镜像改成国内淘宝的,执行以下命令即可

$ npm config set registry https://registry.npm.taobao.org 

$ npm info underscore （如果上面配置正确这个命令会有字符串response）
